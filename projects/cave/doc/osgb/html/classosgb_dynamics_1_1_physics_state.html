void osgbDynamics::PhysicsState::loadState ( )

Load current state from the physics simultation.

This function calls PhysicsData::loadState() for each element of _dataMap. Use this function to take a "snapshot" of the current physics simultaion. osgBullet uses this functionality to save physics state to disk.

void osgbDynamics::PhysicsState::restoreState ( ) const

Restore physics state to the physics simulation.

This function calls PhysicsData::restoreState() for each element of _dataMap. Use this function to reset the physics simulation to a previous state. For example, this function enalbes a "reset" button in your application.

Together, loadState() and restoreState() support the ability for an app to save a physics simulation to disk and restore it later in mid-simulation.
