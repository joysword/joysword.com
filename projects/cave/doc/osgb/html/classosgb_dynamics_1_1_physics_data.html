void osgbDynamics::PhysicsData::loadState ( )

Obtains rigid body information from the physics simulation.

If _body is non-NULL, this function stores the rigid body's current world transform, linear velocity, and angulat velocity in the analogous fields.

void osgbDynamics::PhysicsData::restoreState ( ) const

Restore rigid body information to the physics simultation.

If _body is non-NULL, this function restores the saves _bodyWorldTransform, |c _angularVelocity, and _linearVelocity.
