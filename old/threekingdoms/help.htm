In order to have the computer read the book for you, you need to download and install the Microsoft Agent software. Following are the files you want to install. All files are from Microsoft servers, and are free.

Download and install five files below:
• Microsoft Agent 2.0 (395 KB exe) (don't need it, if you have Windows 2000)
• Speech Engine (English 1 MB exe) (don't need it, if you have Windows 2000)

• Character Merlin (1.8 MB exe) (don't need it, if you have Windows 2000)
• Character Genie (1.6 MB exe) (optional download)
• Character Peedy (3.3 MB exe) (optional download)
