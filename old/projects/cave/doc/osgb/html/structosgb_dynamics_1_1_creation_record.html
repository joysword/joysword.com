Detailed Description

Data record for rigid body construction information.

Fill in this struct and pass it to the createRigidBody() functions to specify rigid body (and collision shape) construction parameters. See the rigid body creation utilities for more information.

This record can be stored in UserData on the rigid body subgraph root node to facilitate saving and restoring physics state.

This struct can be serialized to/from the .osg file format. See the osgdb_osgbDynamics plugin.
