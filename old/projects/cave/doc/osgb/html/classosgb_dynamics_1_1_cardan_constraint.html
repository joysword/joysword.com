Detailed Description

Implements a universal or Cardan constraint, useful for transferring torque from one body to another.

Like the spring constraints, and unlike most Bullet constraints, CardanConstraint requires two rigid bodies. This is a requirement imposed by Bullet.

The axes and anchor point are in world coordinates. _axisA is the rotational axis for _rbA, and _axisB is the rotational axis for _rbB. The anchor point (_point) is the world coordinate point common to both axes.

This class uses btUniversalConstraint internally. Access the Bullet constraint directly with getAsBtUniversal().
