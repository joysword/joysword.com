Detailed Description

Model of a wheel constrained to a vehicle.

The constraint allows up and down spring-like motion along the _springAxis. Rotation around that axis is controlled with setLimit(), and the default is the range -pi/4 to pi/4 radians. The constraint also allows rotation around the _axleAxis. Both axes pass through _point, the center of rotation.

This constraint class, along with the spring constraints, require two rigid bodies. This restriction is imposed by the underlying Bullet support classes.

This class uses btHinge2Constraint internally. Access the Bullet constraint directly with getAsBtHinge2().

void osgbDynamics::WheelSuspensionConstraint::setLinearLimit ( const osg::Vec2 &  linearLimit)

Set the _springAxis linear spring limits.

The spring along the _springAxis acts like a LinearSpringConstraint. Use setLinearLimit() to specify the allowable translation along this axis. Default is -1.0 to 1.0 in world units, with limit position 0.0 corresponding to the initial transformation of the rigid bodies.

Set the _springAxis rotation limits in radians.

In the typical use case of modeling a vehicle wheel, limitRadians indicates the maximum extent the wheel can be turned to the right or left. Limit angle extents are in radians around _springAxis. Default is the range -pi/4 to pi/4.
