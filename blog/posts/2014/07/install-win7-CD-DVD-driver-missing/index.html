安装win7时提示"缺少所需的CD/DVD驱动器设备驱动程序"

2014-07-29 | 伪技术宅 | #windows, #重装

前两天帮一个朋友重装系统，点击了‘现在安装’之后竟然报错，缺少所需的CD/DVD驱动器设备驱动程序。

我是用U盘安装的，不是用的光盘，为什么去寻找CD/DVD驱动器？经研究发现，解决方法竟然是：

把U盘拔了，重新插上

而错误原因也很简单：U盘没有读出来。或者说电脑启动的时候成功地从U盘引导了，但是进入安装界面以后没有能够读取U盘。

事实证明那个USB接口是USB 3.0的，用USB 3.0的接口就是容易出现这个问题。

所以以后安装系统的时候就使用USB 2.0的接口吧。
