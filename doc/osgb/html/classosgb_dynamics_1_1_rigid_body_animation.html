Detailed Description

An update callback to reposition a btRigidBody in the Bullet simulation.

This callback repositions an object within the Bullet simulation. Attach it as an update callback to an OSG MatrixTransform. The MatrixTransform must have an osgbCollision::RefBulletObject< btRigidBody > attached as UserData.
